# React Testimonial Slider

A simple testimonial slider built with React using `useState` and `useEffect`.

## Features

* Automatic slide rotation
* React Hooks (`useState`, `useEffect`)
* Clean and reusable component structure
* Responsive testimonial layout
* Easy to customize content and styling

## Technologies

* React
* JavaScript (ES6+)
* CSS

## Project Structure

```text
src/
├── components/
│   └── Avatar.jsx
├── Avatar.css
└── App.jsx
```

## How It Works

The component stores testimonial data in an array and automatically changes the active testimonial every 7 seconds using `setInterval`.

## Installation

```bash
git clone https://github.com/setareh-kazemi10/react-testimonial-slider.git
cd react-testimonial-slider
npm install
npm run dev
```

## Learning Goals

* Working with React Hooks
* Managing component state
* Using `useEffect` for side effects
* Creating simple sliders without external libraries
